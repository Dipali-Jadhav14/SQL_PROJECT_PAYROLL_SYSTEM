# SQL_PROJECT_PAYROLL_SYSTEM
# Features
- Add new employees with details like name, department, and basic salary.
- Update employee leaves and bonus.
- Generate payslips automatically with gross salary, deductions, and net salary.
- View all employees and their details.
- Uses MySQL database to store all employee and payslip records.

---

# Technologies
- **Python** – Core programming language
- **MySQL** – Database for storing employee and payslip data
- **mysql-connector-python** – Python library to connect with MySQL




Add Employee function– Enter employee name, department, and basic salary.

Show Employees – Displays all employees with their ID, salary, leaves, and bonus.

Update Leaves & Bonus – Modify leaves and bonus for a specific employee using their Employee ID.

Generate Payslip – Automatically calculate gross salary, deductions, and net salary for an employee and save it in the database.

Exit – Close the program.
